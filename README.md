# Лабораторная работа №2
Вариант 28

Вычислить сумму знакопеременного ряда -(|х*n!|)/n!, где х-матрица ранга к (к и матрица задаются случайным образом), n - номер слагаемого. Сумма считается вычисленной, если точность вычислений будет не меньше t знаков после запятой. У алгоритма д.б. линейная сложность. Операция умножения –поэлементная.
